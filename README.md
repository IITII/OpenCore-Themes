## 安装

### Resource
* 替换 EFI > OC > Resources 文件夹
* 或者
* 将 LuckyCrack 文件夹复制至 EFI > OC > Resources > image 下

### config.list
* Misc > Boot > PickerMode > External
* Misc > Boot > PickerAttributes > 129
* Misc > Boot > PickerVariant: LuckyCrack\BigSur
